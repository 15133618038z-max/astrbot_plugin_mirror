# 图片对称反转 — Mirror

> 把一张正常的图片变成左右对称，然后你会发现人类的脸有多不对称。

---

## 🔍 这是什么

一个专门搞抽象的图片处理插件。

发送或引用一张图片，输入「反转」，Bot 会自动把左半边镜像到右半边，生成一张**左右对称**的图。

效果嘛……有的像蝴蝶，有的像克苏鲁，有的直接让人笑出声。

---

## ✨ 功能特性

### 🪞 对称镜像

左半保留，右半镜像。对称之后你会发现：

- 人脸对称 = 恐谷效应
- 风景对称 = 意外好看
- 表情包对称 = 直接封神

### 📷 多种图片来源

| 触发方式 | 示例 |
|---------|------|
| 引用图片 + 反转 | 长按图片 → 回复 → 输入「反转」 |
| 直接带图 + 反转 | 发送图片的同时输入「反转」 |
| 引用表情 + 反转 | 引用自定义表情 → 输入「反转」 |

### 🎞️ 动图支持

引用 GIF 动态表情？没问题，自动截取首帧，静态输出。

### 📦 多图处理

一次最多处理 **3 张**图片。群友排队搞抽象也够用。

### 🧹 自动清理

生成的图片发完就删，不占你服务器空间。临时文件 30 秒后自动清理，每小时还会大扫除一次。

---

## 🚀 使用方式

```
[引用一张图片] 反转
```

或

```
[发送图片的同时] 反转
```

Bot 会回复一张对称图。笑不笑随你。

---

## 🎯 效果说明

```
原图：  [左半] [右半]
结果：  [左半] [左半的镜像]
```

看起来很简单，但效果取决于原图。人脸对称大概率吓人，风景对称可能意外好看，表情包对称基本都是神作。

## ⚠️ 注意事项

1. **GIF 动图**：自动截取首帧，输出为静态图。想看动图对称？做不到。
2. **系统表情**：QQ 内置系统表情没法处理，会被跳过。
3. **图片大小**：太大的图处理会慢，建议 10MB 以内。
4. **并发**：支持多人同时用，群友可以一起搞抽象。
